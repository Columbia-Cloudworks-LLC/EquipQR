**For:** Admin, Owner  
**Last reviewed:** 2026-07-06

::: info Requires
Must be Organization Owner or Admin.
:::

By default, only Owners and Admins can see or edit inventory. The **Parts Access** panel grants two levels of access so the shop keeps moving without opening up admin privileges:

- **Parts Managers** can create, edit, delete, and adjust inventory items.
- **Parts Consumers** can view inventory and use part lookup, but cannot edit items.

Members without either grant cannot access inventory at all.

## 1. Open Inventory → Parts Access

From Inventory, click the **Parts Access** button next to Add Item. On a phone, use the **Update parts access** link at the bottom of the inventory list.

## 2. Add members to a role

Each role section has an **Add** button that opens a searchable member picker with select all / select none / inverse controls. Pick one or more members and confirm — access applies immediately.

## 3. Remove access when needed

Click the trash icon on a member's row in either section and confirm. Their access reverts to none (or to their remaining grant).

::: tip Also available on the Members page
Parts Manager and Parts Consumer toggles also appear per member on **Organization → Members**, so you can review inventory access alongside the rest of a member's permissions.
:::
