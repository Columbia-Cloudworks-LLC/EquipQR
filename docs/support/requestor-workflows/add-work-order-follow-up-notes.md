**For:** Requestor  
**Last reviewed:** 2026-07-03

Use public notes when you need to attach documentation after submitting a work order or after it has been completed.

## 1. Open the work order

Go to **Work Orders** and open the work order you created or requested for your team.

## 2. Add a public note

Scroll to **Notes & Updates** and choose **Add Note**.

- Enter your update (for example, a purchase order number or approval reference).
- You can attach photos when needed.
- Requestors only add **public** notes. Private/internal notes are reserved for managers and technicians.

## 3. Completed work orders

You can still add public notes after a work order is marked **Completed** so billing or approval evidence can be captured later.

Notes cannot be added to **Cancelled** work orders.
