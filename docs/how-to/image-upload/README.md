## 🔧 Technical Requirements

### Supported Image Formats
- JPG/JPEG
- PNG
- GIF

### File Size Limits
- Maximum 10MB per image
- Up to 10 images per note

### Browser Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- Stable internet connection
